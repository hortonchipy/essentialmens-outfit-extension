Features

Fast UI

Simple and clean interface with instant suggestions.

Works Offline

No API or internet connection required.

Smart Suggestions

Predefined outfit combinations for multiple occasions.

Lightweight

Minimal extension size with smooth performance.

How to Install

  1. Download the ZIP file
  2. Extract the folder
  3. Open Chrome browser
  4. Go to chrome://extensions/
  5. Enable Developer Mode
  6. Click Load Unpacked
  7. Select the extracted extension folder
