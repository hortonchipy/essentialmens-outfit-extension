
# EssentialMens Outfit Generator

Modern Chrome extension for outfit inspiration and fashion styling suggestions.

## Features

- Outfit recommendations
- Fashion inspiration
- Simple popup interface
- Visual outfit preview
- Chrome extension support

## GitHub Pages

After enabling GitHub Pages:
https://hortonchipy.github.io/essentialmens-outfit-extension/

## Installation

1. Download ZIP
2. Extract files
3. Open chrome://extensions/
4. Enable Developer Mode
5. Click Load Unpacked
6. Select folder
